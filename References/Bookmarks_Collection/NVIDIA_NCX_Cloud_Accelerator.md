# NVIDIA Cloud Accelerator (NCX)
**Source URL:** https://docs.nvidia.com/ncx/index.html
**Date Fetched:** 2026-04-12

## Overview

NVIDIA Cloud Accelerator (NCX) is a portfolio of open, modular infrastructure software components that cloud partners can use to build and operate NVIDIA-powered AI clouds. The platform draws from NVIDIA's internal AI infrastructure experience and provides composable components spanning infrastructure and platform layers, including lifecycle management, health monitoring, and operational automation.

## Key Software Components

NCX includes several specialized tools:

- **NVIDIA Cloud Functions (NVCF)**: A unified API for scaling inference and simulation workloads across Kubernetes clusters
- **Dynamo**: A distributed inference-serving framework for multi-node data center deployments
- **NVIDIA Fleet Intelligence**: An agent-based service providing GPU health monitoring and predictive failure detection
- **KAI Scheduler**: A Kubernetes scheduler optimized for GPU resource allocation at scale
- **NVSentinel**: Open-source Kubernetes-native GPU monitoring with automated fault remediation
- **AI Cluster Runtime**: Provides validated Kubernetes runtime definitions for reproducible GPU infrastructure
- **NCX Infra Controller**: Handles bare-metal provisioning and secure lifecycle management for multi-tenant GPU infrastructure

The platform provides several guides for cloud partners and ISVs, including software reference guidelines, inference architecture recommendations, and NVIDIA's requirements for AI clouds infrastructure and operations.
