## Architecture Section

The page includes a diagram illustrating UCX's three-layer framework structure.

## Three Core Frameworks

### 1. UC-S (Services)
"Provides basic infrastructure for component-based programming, memory management, and useful system utilities"

### 2. UC-T (Transport)
"Low-level API that expose basic network operations supported by underlying hardware"

### 3. UC-P (Protocols)
"High-level API uses UCT framework to construct protocols commonly found in applications (MPI, OpenSHMEM, PGAS, etc.)"
