# GPUs Can Now Use PCIe-Attached Memory or SSDs to Boost VRAM Capacity

## Overview

The article discusses an emerging technology that allows graphics processors to expand their memory capacity by utilizing PCIe-attached storage devices and solid-state drives, rather than being limited to onboard VRAM.

## Key Technical Details

According to the piece, Panmnesia has developed CXL intellectual property that enables this memory expansion capability. The technology achieves what the article describes as "double-digit nanosecond latency," suggesting minimal performance degradation when accessing external memory.

## Industry Context

The article notes an important qualification: "Provided that AMD, Intel, and Nvidia would like to support CXL for memory expansions." This indicates the technology's viability depends on adoption by major GPU manufacturers.

## Significance

This development addresses a persistent constraint in GPU computing—limited onboard memory—by leveraging existing PCIe infrastructure and CXL standards to create a more flexible memory hierarchy. The approach offers potential benefits for machine learning, scientific computing, and other memory-intensive applications.

The solution represents an engineering alternative to simply increasing physical VRAM on devices, potentially reducing costs while improving flexibility in memory management.
