# New InfiniBand and RoCE Specification Introduces Memory Placement Extensions

**Source URL:** https://www.infinibandta.org/new-infiniband-and-roce-specification-introduces-memory-placement-extensions-to-significantly-reduce-persistent-memory-latency/
**Date accessed:** 2026-03-11

**Published:** August 18, 2021
**Author:** Denise Jarrett-Weeks

## Overview

The InfiniBand Trade Association announced the release of IBTA Specification Volume 1 Release 1.5, introducing significant enhancements to reduce latency in persistent memory applications.

## Key Features

**Memory Placement Extensions (MPE)**
The specification enables implementation of RDMA Flush and RDMA Atomic Write operations, designed to "enhance PMEM performance and reduce overall latency by a factor of 10." Data persistence is guaranteed through RDMA operations over InfiniBand or RoCE interconnects.

**NDR InfiniBand Support**
The update includes support for NDR technology, enabling 400 Gbps speeds on 4-lane port configurations. A 2-lane variant supports 200 Gbps while increasing port density.

**Quality of Service Enhancements**
An updated VL Arbitration mechanism allows users to specify minimum bandwidth on physical ports and control virtualized port arbitration through rate limiting and bandwidth sharing capabilities.

## Target Applications

The specification addresses requirements in:
- Enterprise Data Centers (EDC)
- High Performance Computing (HPC) environments
- Cloud data centers
